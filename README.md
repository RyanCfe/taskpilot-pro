# TaskPilot Pro

**Ranked AI tools and smart prompts for every task.**

TaskPilot Pro is a Chrome extension that helps users choose the right AI tool for what they are trying to do.

Instead of searching things like:

- best AI tool for coding
- best AI tool for PPT
- best AI tool for research papers
- best AI tool for math
- best AI tool for shopping
- best AI tool for trading

TaskPilot Pro organizes tools by task, ranks them, gives you a strong prompt, and opens the selected tool instantly.

## How to install manually

TaskPilot Pro is not on the Chrome Web Store yet, so install it manually:

1. Download this repository as a ZIP.
2. Unzip the folder.
3. Open Chrome.
4. Go to `chrome://extensions/`.
5. Turn on **Developer mode**.
6. Click **Load unpacked**.
7. Select the unzipped TaskPilot Pro folder.
8. Pin the extension and start using it.
