# TaskPilot Pro - Ranked AI Tools

TaskPilot Pro is a Chrome Extension MVP that helps users choose the right AI/tool for a task.

## Features

- Ranked AI/tools by task, not generic list.
- Categories: AI chat/search, coding, PPT, study, math, research papers, books, originality/citations, news, image/design, video/audio, data, business, shopping, trading/finance, productivity, no-code, security launch.
- Prompt Builder for custom high-quality prompts.
- Right-click selected text actions:
  - Explain
  - Summarize
  - Rewrite
  - Research with sources
  - Make study notes
  - Turn into PPT outline
  - Fact-check
  - Originality/citation review
- Launch Guard security checklist.
- Local-only saved prompts/favorites.
- No backend, no tracking, no analytics.

## Run locally

1. Open Chrome.
2. Go to `chrome://extensions/`.
3. Enable Developer Mode.
4. Click `Load unpacked`.
5. Select this folder.
6. Pin the extension.

## Important

The originality/AI-detector section is intended for ethical review: citation gaps, unsupported claims, and improving genuine originality. It is not intended to help users deceive academic or workplace detectors.

## Privacy

Favorites, checklist progress, and right-click temporary prompts are stored locally in Chrome storage. Nothing is sent to any server by this extension.
